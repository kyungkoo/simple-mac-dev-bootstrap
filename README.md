simple-mac-dev-bootstrap
========================

This is simple shell script to install homebrew and formula for development.

Packages
--------
- Homebrew
 - Python 2.x
 - node.js
 - vim
 - tree
 - git
 - gradle
 - tmux

Execute
-------
**Just call ./mac_dev_bootstrap.sh**

Trouble Shooting
----------------
If you got error about permission denied.
**chomd a+x ./mac_dev_bootstrap.sh**
